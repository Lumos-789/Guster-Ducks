# Wallpaper（壁纸轮播）

Pull fresh wallpapers from curated sources, rotate them on a schedule.

## Acceptance

- Fetches wallpapers from Unsplash, Bing Daily, or local folder
- Sets macOS desktop wallpaper via script or `osascript`
- Supports timed rotation (every 30 min, daily, on wake)
- No manual intervention after initial setup
