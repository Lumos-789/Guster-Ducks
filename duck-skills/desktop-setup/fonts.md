# Fonts（字体调性）

Set up fonts that look good and read well across terminal, editor, and system UI.

## Acceptance

- Recommends and installs a cohesive font stack (mono for code, sans for UI)
- Configures terminal and editor to use them
- Handles CJK fallback so Chinese renders cleanly
- Output is a one-command install or a copy-paste config block
